## 🚀 Release Notes

This checkpoint represents the absolute ceiling of what a **fully frozen** ViT-Base backbone can achieve on this dataset. Training naturally early-stopped at Epoch 11 as the frozen features reached their representational limit without overfitting.

### Key Improvements in this Release:
* **Test-Time Augmentation (TTA)**: Introduced Horizontal Flip TTA evaluation, boosting the final submitted score to `0.5169`.
* **Multi-Scale Augmentations**: Replaced static resized crops with aggressive multi-scale random cropping (`ScaleJitter` equivalents) to force the UPerNet head to learn scale invariance.
* **Loss Rebalancing**: Shifted Focal+Dice weighting to `1.0 * Focal + 0.5 * Dice` to reduce over-confidence artifacts dynamically introduced by the dice term on small batches.

### Architecture Details
* **Backbone**: `facebook/dinov2-base` (768-dim embeddings) — **Fully Frozen**
* **Segmentation Head**: `UPerNetHead`
* **Input Resolution**: `644 × 364`
* **Effective Batch Size**: `4` (Gradient Accumulation over 2 steps)

### Note on Checkpoint File Size
File size remains at **39 MB** because the backbone is fully frozen. The optimizer state dictionary only tracks momentum and variance for the parameters of the UPerNet head.
